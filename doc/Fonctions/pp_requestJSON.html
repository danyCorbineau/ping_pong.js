pp_requestJSON

Permet d'appeler une fonction suite à l'appel d'une page web.

pp_requestJSON(url,data,callback)

Arg url String L'URL à appeler.
Arg data FormData Objet FormData, voir pp_createFormData.
Arg callback fonction(valeur) La fonction appelé à la réception des données de la page avec valeur, les données de la page (JSON Object).
Return
