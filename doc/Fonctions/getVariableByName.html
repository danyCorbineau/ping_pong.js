getVariableByName

Permet d'obtenir une variable des données JSON avec son nom.

getVariableByName(vName, jsonData)

Arg vName String Nom de la variable.
Arg jsonData JSON Object L'objet JSON en lien avec le template.
Return value (Number, String, Object, Array) La valeur de la variable.
