getNextEndRequest

Permet de récupérer la requête de fin correspondant à la requête envoyé.

getNextEndRequest(request,endtext,htmlStream)

Arg request {start: Int, end: Int, text: String} La requête de début.
Arg endtext String Le texte de la requetée de fin à trouver.
Arg htmlStream String Le template dans lequel la recherche est effectuée.
Return endRequest {start: Int, end: Int, text: String} La requête de fin ou null.
